# CloudWatch Logs Group

Sets up and configures CloudWatch Logs, using the regional KMS key generated by the cloudwatchlogs_regional module for encryption.

# AWS IAM Permissions

The following permissions are required to use this module, shown as a Policy snippet in JSON.

```json
{
  "Effect": "Allow",
  "Action": [
    "cloudwatch:*",
    "logs:*"
  ],
  "Resource": "*"
}
```
