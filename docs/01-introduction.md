# Apres Foundations

The Apres Foundations documentation helps you get started with the Apres
Foundations framework. It provides a step-by-step guide to setting up your
AWS Organization and deploying your services using Apres Foundations.

## Apres Foundations Overview

Setting up a new AWS Organization with Apres Foundations has the following
high-level steps:

1. Set up an AWS Organization - [Part 1](04-getting-started-part-1-aws-bootstrap.md)
2. Set up Apres Foundations - [Part 2](05-getting-started-part-2-apres-foundations.md)
3. Deploy your services - [Part 3](06-getting-started-part-3-deploy-your-services.md)
